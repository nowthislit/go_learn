# 01 - Hello World（入门基础）

## 学习目标
- 了解Go语言的特点和应用场景
- 掌握Go开发环境配置
- 理解Go模块(module)的概念
- 编写并运行第一个Go程序

## 文件说明

### main.go
第一个Go程序，输出 "Hello, Go!"

### 核心概念
1. **package main** - 可执行程序的入口包
2. **import** - 导入标准库或其他包
3. **func main()** - 程序入口函数

## 常用命令
```bash
go run main.go      # 编译并运行
go build            # 编译生成可执行文件
go fmt              # 格式化代码
go mod init         # 初始化模块
```

## 练习题
1. 修改程序输出你的姓名和年龄
2. 尝试使用 `go fmt` 格式化代码
3. 使用 `go build` 生成可执行文件并运行
