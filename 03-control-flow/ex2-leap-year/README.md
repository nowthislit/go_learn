## 闰年规则
1. 能被4整除 **且** 不能被100整除，**或者**
2. 能被400整除

```
(year%4 == 0 && year%100 != 0) || year%400 == 0
```

## 测试用例
- 2000: 是（能被400整除）
- 2020: 是（能被4整除，不能被100整除）
- 1900: 否（能被4和100整除，但不能被400整除）
- 2024: 是
- 2023: 否
